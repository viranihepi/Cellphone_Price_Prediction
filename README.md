# Cellphone_Price_Prediction
ML classification project for mobile price prediction.

Dataset : Conatin mobile specifications and price_range as target(cellphone_dataset.xls)

Steps of Project: 
  - Basic checks & EDA
  - Preprocessing (Capping outliers, Scaling, Train-test-split)
  - Built and evaluated 7 ML models
  - Selected best model
  - Final report and conclusion

Result : Logistic Regression with 0.97% accuracy.

How to Run: 
Using Git Clone Method..
  - Copy the repository URL from GitHub.
  - Open Command Prompt (CMD).
  - Clone the repository using:  git clone "copied url".
  - Open Jupyter Notebook and navigate to the cloned folder.
  - Open the .ipynb file and run cells step by step.
